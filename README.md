# README

Code for the manuscript by **Yuan Tao, Erick Delage, and Huifu Xu**, *“Risk-averse Decision Making with Contextual Information: Model, Sample Average Approximation, and Kernelization”* (arXiv:2502.16607, 2025).

arXiv: `https://arxiv.org/abs/2502.16607`

## Contents

- The files `Code_Section_6_1_..._m` contain the code for the **newsvendor problem** in Section 6.1 of the paper.
- The file `Code_Section_6_2.m` contains the code for the **portfolio selection problem** in Section 6.2 of the paper.

## Requirements

The code requires the following software/packages:

- **YALMIP**
- **Gurobi**
- **MOSEK**
